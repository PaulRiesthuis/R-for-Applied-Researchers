1  Why R?

Before we start learning how to program in R, it is useful to first ask: Why learn R at all? In my view, R opens up an enormous range of possibilities—not only for data analysis, but also for data visualization, simulation, reproducible research, open science, or writing an Introduction to R book. In this short introductory chapter, I will highlight a few of the reasons why I think learning R is worthwhile.

1.1 Data analyses

There are many point-and-click programs for statistical analysis such as SPSS, Minitab, Jamovi, and JASP. These are useful tools, but R has several important advantages.

  • R is free and open source, meaning anyone can use it without licensing costs.

  • R is incredibly powerful and flexible. With just a few lines of code, you can run very simple analyses—or extremely advanced ones.

  • R gives you full control. Instead of relying on menus, you can precisely specify your analyses and make your workflow transparent and reproducible.

For example, here is an independent samples t-test with simulated data:

1.2 Data visualization

R is also an outstanding tool for data visualization. With packages such as ggplot2, you can design clean, publication-ready figures and fully customize them. The range of available visualization packages is enormous, making it possible to create almost any kind of figure you can imagine.

# Load package
library(ggplot2)
# Simulate data
sample_df <- data.frame(
  group = factor(rep(letters[1:3], each = 10)),
  value = rnorm(30)
)

# Create df with means
group_means_df <- setNames(
  aggregate(value ~ group, sample_df, mean),
  c("group", "group_mean")
)

# Create plot
vis <- ggplot(data = sample_df, mapping = aes(x = group, y = value)) +
  geom_point() +
  geom_point(
    mapping = aes(y = group_mean), data = group_means_df,
    colour = 'red', size = 3
  )+
  theme_classic()

1.4 Open science

One of the strongest reasons to learn R is that it supports open, transparent, and reproducible science. Using R Markdown or Quarto, you can combine text, code, results, and figures in a single document. This means others can see exactly how analyses were conducted—and can reproduce or adapt them easily. You can even write entire research papers, reports, or books with these tools.

1.5 Fun (and Surprisingly Useful) Extras

  • Create QR codes

    library(qrcode)

  • Build interactive dashboards and Shiny apps

  • Generate geographical maps

  • Scrape and analyze web data

  • …and much more.
